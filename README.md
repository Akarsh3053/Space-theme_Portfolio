 # Setup and run:
- Step 1: Clone the repository -
```bash
git clone https://github.com/Akarsh3053/Portfolio
``` 
- Step 2: Open the terminal in the project directory and run the following command to install dependencies -
```bash
npm install
```
- Step 3: Then run the following command to launch the local server -
```bash
npm run dev
```
